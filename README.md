## Installation: 
1. Download archive with project.
2. Unzip it.
3. Install pods from podfile
4. Open with Xcode.
5. Install on iPhone with iOS 15.5+

#### Stack: Swift, UIKit, CoreData, URLSession, CocoaPods (Kingfisher, youtube_ios_player_helper)
